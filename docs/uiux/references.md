## 3. 레퍼런스별 상세 분석

### 3.1 Melbourne Food & Wine Festival ⭐ 메인 레퍼런스

**전체 톤**
- 밝은 배경 + 따뜻한 컬러 액센트
- 정보 제공 중심, 스캔하기 쉬운 구조
- 정보 밀도: 중간 (카드 기반)

**참고할 요소**
| 요소 | 설명 | FesMate 적용 |
|------|------|-------------|
| **상태 뱃지** | "Selling Fast", "Sold Out", "Running All Festival" | LIVE, SOON, ENDED 뱃지 |
| **이벤트 카드** | 이미지 + 제목 + 날짜 + 장소 | 행사 카드 기본 구조 |
| **카테고리 섹션** | Featured → Global Dining → Special Events | 홈 섹션 구조 |
| **가로 스크롤** | 카테고리별 카드 캐러셀 | 홈 섹션 레이아웃 |

**컬러 팔레트**
- 배경: 화이트/오프화이트
- 텍스트: 블랙/다크 그레이
- 액센트: 이벤트 포스터 컬러 활용

### 3.2 SOIL (soil-net.jp) ⭐ 메인 레퍼런스

**전체 톤**
- 일본 미니멀리즘의 정수
- 크림 화이트 배경
- 정보 밀도: 낮음 (여백 중시)

**참고할 요소**
| 요소 | 설명 | FesMate 적용 |
|------|------|-------------|
| **번호 매김 네비** | 1. HOME, 2. WORKS, 3. JOURNAL... | 사이드 네비게이션 (옵션) |
| **타이포그래피 믹스** | Serif (제목) + Sans (본문) | 타이포 계층 구조 |
| **페이지 전환** | 부드러운 fade + slide (200-300ms) | 탭/페이지 전환 |
| **작품 카드** | 세로 비율, 상태 라벨, 호버 "View More" | 행사 카드 스타일 |
| **스크롤 효과** | 요소별 stagger 등장 | 섹션 로딩 애니메이션 |

**마이크로 인터랙션**
- 호버: 부드러운 색상 전환
- 전환: 120-200ms ease-out
- 스크롤: 순차 등장 (stagger 50ms)

## 5. 컬러 팔레트 제안 (라이트 테마)

### 5.1 기본 팔레트

```css
/* 배경 */
--bg-primary: #FAFAF8;      /* 웜 화이트 (메인) */
--bg-secondary: #F5F5F0;    /* 약간 어두운 웜 화이트 */
--bg-surface: #FFFFFF;      /* 카드, 모달 */

/* 텍스트 */
--text-primary: #1A1A1A;    /* 거의 블랙 */
--text-secondary: #6B6B6B;  /* 미디엄 그레이 */
--text-muted: #9CA3AF;      /* 라이트 그레이 */

/* 보더 */
--border-default: #E5E5E5;  /* 기본 보더 */
--border-hover: #D1D1D1;    /* 호버 보더 */

/* 상태 컬러 */
--status-live: #EF4444;     /* 레드 - LIVE */
--status-soon: #F59E0B;     /* 앰버 - SOON */
--status-ended: #9CA3AF;    /* 그레이 - ENDED */
--status-success: #10B981;  /* 그린 - 성공/확인 */

/* 액센트 (선택) */
--accent-primary: #7C3AED;  /* 퍼플 (브랜드 옵션 A) */
--accent-primary: #FF6B35;  /* 오렌지 (브랜드 옵션 B) */
```

### 5.2 섀도우

```css
/* 소프트 섀도우 (5-8% 불투명도) */
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.07);
--shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1);

/* 호버 시 섀도우 증가 */
--shadow-hover: 0 8px 25px rgba(0, 0, 0, 0.1);
```

### 5.3 라운드 코너

```css
--radius-sm: 8px;    /* 버튼, 칩 */
--radius-md: 12px;   /* 작은 카드 */
--radius-lg: 16px;   /* 메인 카드 */
--radius-full: 999px; /* 원형 (아바타, 뱃지) */
```

## 6. 기술 스택 참고

| 레퍼런스 | 기술 | 비고 |
|----------|------|------|
| SOIL | Next.js + CSS | 부드러운 전환, 미니멀 |
| Melbourne F&W | WordPress + Custom | 카드 레이아웃 |
| Eventbrite | React | 필터/검색 UX |
| Wilde Weide | Next.js + Three.js | 3D 요소 (심화) |
| KIFF Aarau | - | 대시보드 타일 |

**FesMate 적용:**
- `framer-motion` - 탭/페이지 전환, 모달 애니메이션
- `tailwindcss-animate` - 기본 애니메이션 유틸리티
- CSS transitions - 호버/상태 변경 (150-200ms)
- CSS variables - 테마 토큰 관리

**성능 최적화 (vercel-react-best-practices 스킬 적용):**
- `bundle-dynamic-imports` - 무거운 컴포넌트 동적 로딩 (타임테이블, 에디터)
- `bundle-barrel-imports` - 직접 import (barrel 파일 피하기)
- `async-suspense-boundaries` - 섹션별 Suspense 스트리밍
- `rerender-memo` - EventCard, FeedItem 등 리스트 아이템 memo
- `rendering-content-visibility` - 긴 피드 리스트 최적화
