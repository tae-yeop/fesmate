## 1. 배경 및 목표

### 1.1 프로젝트 성격
FesMate는 **행사 정보 허브 + 커뮤니티**가 혼재된 서비스입니다.
- 정보 제공: 행사 일정, 타임테이블, 예매 알림
- 실시간: LIVE 모드, 현장 제보
- 커뮤니티: 동행, 택시팟, 후기
- 기록: 다녀옴, 리뷰, 영상

### 1.2 개선 목표
- **정보 중심**: 복잡하지 않게 핵심 정보 전달
- **라이트 테마**: 웜 화이트 배경 + 소프트 섀도우
- **은은한 전환**: 120~200ms 미세 애니메이션
- **레퍼런스 기반**: Melbourne F&W 구조 + SOIL 톤 + Eventbrite UX

### 1.3 타겟 유저
- 20~30대 공연/페스티벌/전시 팬
- 모바일 퍼스트 (PWA 지향)

### 1.4 주요 레퍼런스
| 레퍼런스 | 적용 포인트 |
|----------|-------------|
| **Melbourne Food & Wine Festival** | 섹션 구조, 상태 뱃지, 카드 레이아웃 |
| **SOIL** | 톤, 타이포 믹스, 부드러운 전환 |
| **Eventbrite** | 카테고리 필터, 검색 UX |

## 3. 제안 홈 구조 (TO-BE)

### 3.1 섹션 구조

```
홈 (vNext)
├── [1] 오늘의 상태 히어로
│   └── 요약 카드 3개: LIVE 수 / 오늘 일정 / 커뮤니티 새글
├── [2] LIVE Now 스트립
│   └── LIVE 행사 카드 (세로 2~3개, 진행바 + CTA)
├── [3] 곧 시작 / 오늘 (가로 스크롤)
│   └── EventCard + 상태 뱃지 (TODAY / SOON)
├── [4] 추천 행사 (2열 그리드)
│   └── EventCard + 포스터 강조 + 찜 수 서브텍스트
├── [5] 커뮤니티 하이라이트
│   └── 통일된 카드 스타일 + 상태/응답수/행사명
├── [6] Social Pulse (통합)
│   └── 친구 활동 + 리더보드 탭형 전환
└── [7] 빠른 액션 바 (옵션)
    └── 글쓰기 / 행사 탐색 / MyFes
```

### 3.2 섹션별 상세

#### [1] 오늘의 상태 히어로
- **목적**: 한눈에 오늘 상황 파악
- **구성**: 3개 요약 카드 (LIVE 행사 수 / 오늘 일정 / 커뮤니티 새글)
- **스타일**: soft gradient 배경 + 텍스트 중심
- **Micro**: 숫자 카운트 0.2s 상승 애니메이션

#### [2] LIVE Now 스트립
- **목적**: 진행 중인 행사 강조
- **구성**: 세로 카드 리스트 (최대 3개)
- **카드 요소**: 포스터 썸네일 / LIVE 뱃지 / 제목 / 장소 / 허브 CTA
- **Micro**: LIVE 뱃지 1.5s pulse (opacity 0.7→1)

#### [3] 곧 시작 / 오늘
- **목적**: 7일 내 행사 빠른 탐색
- **구성**: 가로 스크롤 EventCard
- **추가**: TODAY / SOON 상태 뱃지
- **Micro**: 카드 hover/press scale 0.98 + shadow 증가

#### [4] 추천 행사
- **목적**: 인기/관심 기반 추천
- **구성**: 2열 그리드 EventCard
- **추가**: 포스터 강조, "찜 N명" 서브텍스트
- **스타일**: glass 카드 + soft shadow

#### [5] 커뮤니티 하이라이트
- **목적**: 활발한 모집/글 노출
- **구성**: 글 타입 칩 (동행/택시/밥) + 카드 리스트
- **카드 요소**: 타입 뱃지 / 상태 (모집중) / 응답 수 / 행사명 / 내용 스니펫
- **Micro**: 카드 등장 120ms fade+slide

#### [6] Social Pulse
- **목적**: 친구 활동 + 리더보드 통합
- **구성**: 탭 전환 (최근 활동 / 상위 멤버)
- **Micro**: 탭 전환 150ms fade

#### [7] 빠른 액션 바 (옵션)
- **목적**: 주요 기능 빠른 진입
- **구성**: 3개 아이콘 버튼 (글쓰기 / 탐색 / MyFes)
- **위치**: 히어로 하단 또는 플로팅

## 4. 마이크로 인터랙션 가이드

| 요소 | 효과 | 시간 | 비고 |
|------|------|------|------|
| 카드 hover/press | scale 0.98 + shadow 8→10% | 150ms | ease-out |
| LIVE 뱃지 | pulse (opacity 0.7→1) | 1.5s | infinite |
| 숫자 카운트 | 0→N 상승 | 200ms | ease-out |
| 섹션 등장 | fade + slide-up | 120ms | stagger 50ms |
| 탭 전환 | fade + underline slide | 150ms | ease-in-out |
| 새 글 하이라이트 | bg highlight | 150ms | fade-out 2s |

## 5. 톤 가이드 (홈용) - 라이트 테마

### 5.1 컬러 팔레트

```css
/* 배경 */
--bg-primary: #FAFAF8;      /* 웜 화이트 (메인) */
--bg-secondary: #F5F5F0;    /* 약간 어두운 웜 화이트 */
--bg-surface: #FFFFFF;      /* 카드, 모달 */

/* 텍스트 */
--text-primary: #1A1A1A;    /* 거의 블랙 */
--text-secondary: #6B6B6B;  /* 미디엄 그레이 */
--text-muted: #9CA3AF;      /* 라이트 그레이 */

/* 보더 */
--border-default: #E5E5E5;
--border-hover: #D1D1D1;

/* 상태 컬러 */
--status-live: #EF4444;     /* LIVE - 레드 */
--status-soon: #F59E0B;     /* TODAY/SOON - 앰버 */
--status-ended: #9CA3AF;    /* ENDED - 그레이 */
--status-recruiting: #10B981; /* 모집중 - 그린 */
```

### 5.2 Shape
- **Card radius**: 16px
- **Chip radius**: 999px (pill)
- **Button radius**: 12px

### 5.3 Shadow (소프트 - 5~10% 불투명도)
- **기본**: `0 1px 2px rgba(0,0,0,0.05)`
- **Card**: `0 2px 8px rgba(0,0,0,0.07)`
- **Hover**: `0 4px 12px rgba(0,0,0,0.10)`
- **Elevated**: `0 8px 24px rgba(0,0,0,0.12)`

### 5.4 Typography
- **섹션 제목**: 18~20px, bold (옵션: Serif 폰트)
- **카드 제목**: 14~16px, medium
- **본문**: 14px, regular
- **보조/캡션**: 12px, muted (#6B6B6B)
