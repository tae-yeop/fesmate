# FieldNote 네이밍 결정 문서

> 가이드 탭의 브랜드 네이밍 검토 과정과 최종 결정

## 배경

기존 "가이드" 탭 명칭이 딱딱하고 설명서/매뉴얼 느낌이 강해 브랜딩 관점에서 재검토가 필요했습니다.

### 요구사항
1. **능동적 참여** - 팬이 직접 만들고 공유하는 느낌
2. **현장감** - 공연장에서 쓰는 생생한 정보
3. **커뮤니티** - 함께 만드는 집단지성
4. **확장성** - 콘서트뿐 아니라 뮤지컬/클래식/전시까지
5. **FesMate 브랜드와 조화**

## 검토 과정

### 1차 후보군

| 카테고리 | 후보 | 평가 |
|----------|------|------|
| 동작/액션 | Vibe, Hype, Sync | Vibe가 가장 감각적이나 추상적 |
| 장소/공간 | Stage, Scene, Spot | Scene이 문화적이나 영화 느낌도 |
| 지식/문화 | Wiki, Lore, Know | Wiki가 집단지성 명확 |
| 감성/경험 | Ready, Prep | 공연 전 설렘, 영어라 거리감 |

**1차 결론**: 단일 단어로는 의미 전달 부족, 합성어 방향으로 전환

### 2차 후보군: 합성어

| 후보 | 조합 | 장점 | 단점 |
|------|------|------|------|
| VibeWiki | Vibe+Wiki | 감각적+집단지성 | 추상적, 4음절 |
| LiveWiki | Live+Wiki | 직관적, 공연 직접 지칭 | 평범할 수 있음 |
| Fanlore | Fan+Lore | 2음절, 팬덤 전승 | 기존 사이트와 중복 |
| Livedex | Live+dex | 도감 느낌, 수집욕 | dex가 생소 |
| Crowdex | Crowd+dex | 집단성 강조 | 발음 어색 |
| Stagedex | Stage+dex | 무대 도감 | 4음절 |

### 3차 후보군: 최종 4개

사용자 제안을 바탕으로 4개 후보 심층 비교:

| 후보 | 발음 | 의미 |
|------|------|------|
| 라이브위키 (LiveWiki) | 4음절 | 라이브 공연의 집단지성 |
| 라이브노트 (LiveNote) | 4음절 | 라이브 공연의 기록 |
| 필드위키 (FieldWiki) | 4음절 | 현장 정보의 집단지성 |
| 필드노트 (FieldNote) | 4음절 | 현장 기록, 탐험가의 노트 |

### 비교 매트릭스

| 기준 | 라이브노트 | 필드노트 | 필드위키 | 라이브위키 |
|------|:----------:|:--------:|:--------:|:----------:|
| 직관성 | ★★★★ | ★★★ | ★★★ | ★★★★★ |
| 집단지성 느낌 | ★★ | ★★ | ★★★★ | ★★★★★ |
| 현장감 | ★★★★ | ★★★★★ | ★★★★ | ★★★★★ |
| 세련됨 | ★★★ | ★★★★★ | ★★★ | ★★★ |
| 팬문화 적합 | ★★★ | ★★★ | ★★★ | ★★★★ |
| 확장성 | ★★★★ | ★★★★ | ★★★ | ★★★★★ |
| 브랜드 고유성 | ★★★ | ★★★★ | ★★★ | ★★★ |

### 순위 및 근거

1. **라이브위키**: 가장 직관적, 집단지성 명확, 안전한 선택
2. **필드노트**: 가장 세련됨, 고급스러움, 차별화 포인트
3. **라이브노트**: 무난하나 집단지성 느낌 약함
4. **필드위키**: 공연과의 연결고리 부족

## 최종 결정: FieldNote (필드노트)

### 선정 이유

1. **세련된 톤앤매너**
   - 탐험가/연구자의 현장 기록 느낌
   - "현장에서 직접 수집한 생생한 정보" 뉘앙스
   - 다른 팬앱과 차별화되는 고급스러움

2. **확장성**
   - 전시/클래식 관객에게 특히 어필
   - 학술적/문화적 깊이감
   - FesMate의 프리미엄 포지셔닝 강화

3. **브랜드 스토리**
   - "팬들이 현장에서 직접 기록하고 공유하는 노트"
   - 집단지성을 "필드 리서치" 메타포로 표현

### 고려사항

- "노트"가 개인 메모 느낌이라 집단지성 표현 약함
  → UI/UX에서 "함께 만드는" 요소 강조 필요
- K-pop 팬덤과 톤이 다를 수 있음
  → 콘텐츠와 비주얼로 친근함 보완

## 적용 계획

### 네비게이션
```
홈 | 탐색 | FieldNote | 커뮤니티 | MyFes
```

### 라우팅 구조
| 경로 | 설명 |
|------|------|
| `/fieldnote` | 필드노트 홈 |
| `/fieldnote/artist/[id]` | 아티스트 상세 |
| `/fieldnote/call/[songId]` | 콜가이드 뷰어 |
| `/fieldnote/call/[songId]/edit` | 콜가이드 에디터 |

### 서브 콘텐츠 네이밍 (안)

| 현재 | FieldNote 내 명칭 (안) |
|------|------------------------|
| 호응법/콜가이드 | Call & Response, 호응 |
| 셋리스트 | Setlist |
| 아티스트 | Artist |
| 관람 팁 | Tips |

## 향후 발전 방향

### 브랜드 확장
- **FieldNote by FesMate**: 서브 브랜드로 독립 가능성
- 웹/앱 분리 시 별도 랜딩 페이지 고려

### 비주얼 아이덴티티
- 노트북/수첩 메타포 활용
- 손글씨/스케치 느낌의 UI 요소
- 스탬프/스티커 같은 기여자 표시

### 게이미피케이션
- "필드노트 기여자" 배지
- 아티스트별 노트 완성도 표시
- 기여 랭킹 시스템

### 커뮤니티 용어
- 기여자 → "필드 에디터" 또는 "노트 메이커"
- 검증됨 → "현장 검증 완료"
